# DynamicArray Implementation

This folder contains a clean, minimal, exam-friendly implementation of a `DynArray<T>` structure, including:

- Dynamic resize / reserve / shrink-to-fit
- Push / pop / erase (single + range)
- Copy semantics
- Move semantics
- Random-access iterator
- Reverse iterator
- Const access
- Complete test suite (`tests.cpp`)

This implementation follows a clear and educational format:
- Header implementations (`dynarray.hpp`)
- Iterator implementations (`dynarray-iterators.hpp`)
- Tests (`tests.cpp`)

A lightweight but complete alternative to heavy academic versions.
